# DeciOps 智策运营大脑

**AI推演，你来拍板 | Let AI reason. You rule.**

## 项目简介

DeciOps 是一个面向企业运营人员的决策支持与审核闭环系统。基于数据中台和智能体中台，自动生成可审核的运营决策。

## 架构

```
┌─────────────────────────────────────────────────────────────┐
│                        前端 UI                              │
│                  (React + Ant Design)                       │
└─────────────────────────┬───────────────────────────────────┘
                          │ HTTP/WebSocket
┌─────────────────────────▼───────────────────────────────────┐
│                    后端 BFF 层                               │
│              (Spring Boot / Go Gin)                         │
└─────────────────────────┬───────────────────────────────────┘
                          │
┌─────────────────────────▼───────────────────────────────────┐
│                 决策分析引擎                                  │
│           (场景触发 → 特征提取 → Agent调用                    │
│            → 决策合成 → 冲突检测)                            │
└─────────────────────────┬───────────────────────────────────┘
                          │
          ┌───────────────┼───────────────┐
          ▼               ▼               ▼
    ┌──────────┐   ┌──────────┐   ┌──────────┐
    │ 数据中台 │   │智能体中台│   │PostgreSQL│
    └──────────┘   └──────────┘   └──────────┘
```

## 模块

| Package | Description |
|---------|-------------|
| `@deciops/common` | 公共类型定义和工具函数 |
| `@deciops/engine` | 决策分析引擎微服务 |
| `@deciops/backend-bff` | 后端 BFF API 服务 |
| `@deciops/frontend` | React 前端应用 |
| `@deciops/deploy` | Docker Compose 和 Helm 部署配置 |

## 快速开始

```bash
# 安装依赖
pnpm install

# 本地开发启动
pnpm deploy:local

# 构建所有模块
pnpm build
```
