# @deciops/common

DecisionOps TypeScript common library providing shared types, utilities, and core functionality for decision management.

## Installation

```bash
npm install @deciops/common
```

## Usage

```typescript
import { DecisionCard, computeConfidence, detectConflicts, validateDecisionCard } from '@deciops/common';
```

## Structure

- `types/` - TypeScript interfaces for DecisionCard, ScenarioConfig, ExecutionTask, AuditRecord
- `utils/` - Utility functions for confidence calculation, conflict detection, and validation
